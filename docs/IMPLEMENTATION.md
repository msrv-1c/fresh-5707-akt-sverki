# Реализация

1. Заимствовать документ и общий модуль печати акта (имя уточнить в выгрузке БП).
2. Расширить макет: параметры `НомерДокумента`, `ДатаДокумента`, `Контрагент`.
3. `ПредставлениеИмениФайлаАктаСверки` — в обработчике имени файла при выгрузке в PDF.
4. Брендинг — футер, если `ГСК5707_ПоказыватьБрендинг`.
